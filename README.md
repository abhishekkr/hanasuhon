
## Vachak

* usages

```
## show help usage
go run vachak.go -h

## as of now prints all pages at once
go run vachak.go -epub /path/to/my.epub

## for Text To Speech enabled; it's slow as of now
go run vachak.go -tts -epub /path/to/my.epub
```
